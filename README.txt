This lightweight plugin integrates BigBlueButton functionality into Wordpress.

== Description ==

[BigBlueButton](https://bigbluebutton.org/ "BigBlueButton") is an open source web conferencing system. This plugin integrates BigBlueButton into WordPress allowing bloggers to create and manage meetings rooms to interact with their readers. It was developed and is maintained by <a href="https://blindsidenetworks.com/" target="_blank">Blindside Networks</a>.

For more information on setting up your own BigBlueButton server or for using an external hosting provider visit [https://bigbluebutton.org/support](https://bigbluebutton.org/support "https://bigbluebutton.org/support").
